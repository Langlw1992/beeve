## 键盘交互 (Keyboard Support)

作为基于 Zag.js 的组件，DatePicker 提供完整的无障碍键盘支持：

- <kbd>Enter</kbd> / <kbd>Space</kbd>: 打开日历
- <kbd>Arrow Keys</kbd>: 在日历中导航
- <kbd>PageUp</kbd> / <kbd>PageDown</kbd>: 切换月份
- <kbd>Shift</kbd> + <kbd>PageUp</kbd> / <kbd>PageDown</kbd>: 切换年份
- <kbd>Main</kbd>: 将焦点移动到当前选定日期
- <kbd>Esc</kbd>: 关闭日历

## API 参考

### DatePicker

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `size` | `sm` \| `md` \| `lg` | `md` | 组件尺寸 |
| `error` | `boolean` | `false` | 是否显示错误状态 |
| `placeholder` | `string` | - | 占位文本 |
| `label` | `string` | - | 标签文本 |
| `class` | `string` | - | 自定义类名 |
| `numOfMonths` | `number` | `1` | 显示的月份数量 |

更多高级属性继承自 Zag.js DatePicker Context。
