Switch 组件用于在两种状态（开启/关闭）之间进行切换。

## 何时使用

- 需要表示开关状态/两种状态之间的切换时
- 相较于 Checkbox，Switch 在视觉上更强调"立即生效"的状态改变

## 基础用法

最简单的用法。

<ComponentPreview>
  <SwitchBasic client:only="solid-js" />
</ComponentPreview>

```tsx
import { Switch } from '@beeve/ui'

<Switch>飞行模式</Switch>
```

## 尺寸

提供三种尺寸：`sm`（小）、`md`（中，默认）、`lg`（大）。

<ComponentPreview>
  <SwitchSizes client:only="solid-js" />
</ComponentPreview>

```tsx
<Switch size="sm">Small</Switch>
<Switch size="md">Medium</Switch>
<Switch size="lg">Large</Switch>
```

## 禁用状态

设置 `disabled` 属性来禁用开关。

<ComponentPreview>
  <SwitchDisabled client:only="solid-js" />
</ComponentPreview>

```tsx
<Switch disabled>禁用</Switch>
<Switch disabled checked>禁用且选中</Switch>
```

## 受控模式

通过 `checked` 和 `onChange` 属性来控制开关状态。

<ComponentPreview>
  <SwitchControlled client:only="solid-js" />
</ComponentPreview>

```tsx
import { createSignal } from 'solid-js'
import { Switch } from '@beeve/ui'

export default function App() {
  const [checked, setChecked] = createSignal(true)

  return (
    <Switch checked={checked()} onChange={setChecked}>
      显示通知
    </Switch>
  )
}
```

## API

### Props

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| checked | `boolean` | - | 指定当前是否选中 |
| defaultChecked | `boolean` | `false` | 初始是否选中（非受控模式） |
| disabled | `boolean` | `false` | 是否禁用 |
| size | `'sm' \| 'md' \| 'lg'` | `'md'` | 开关尺寸 |
| name | `string` | - | input 的 name 属性 |
| value | `string` | - | input 的 value 属性 |
| id | `string` | - | input 的 id 属性 |
| onChange | `(checked: boolean) => void` | - | 变化时的回调函数 |
| class | `string` | - | 自定义类名 |
