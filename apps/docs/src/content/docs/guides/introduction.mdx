## 什么是 Beeve UI？

Beeve UI 是一个基于 [SolidJS](https://www.solidjs.com/) 构建的现代化组件库，专为构建高性能 Web 应用而设计。

### 为什么选择 SolidJS？

- **细粒度响应式**：SolidJS 的响应式系统仅更新真正变化的 DOM 部分
- **编译时优化**：没有虚拟 DOM 开销，直接编译为高效的原生 JS
- **熟悉的语法**：类似 React 的 JSX 语法，上手简单
- **更小的体积**：运行时只有 ~7KB gzip

### 设计原则

1. **简洁优先**：API 设计简洁明了，减少心智负担
2. **类型安全**：完整的 TypeScript 类型定义
3. **可组合**：组件可以灵活组合使用
4. **无障碍**：遵循 WAI-ARIA 规范

### 技术栈

| 技术 | 用途 |
|------|------|
| SolidJS | UI 框架 |
| TailwindCSS v4 | 样式系统 |
| tailwind-variants | 组件变体 |
| TypeScript | 类型系统 |
