Beeve UI 是一套基于 SolidJS 的现代化组件库，专为构建高质量的用户界面而设计。

## 设计理念

### 响应式优先

组件充分利用 SolidJS 的细粒度响应式系统，确保最佳的运行时性能。

### 无障碍支持

基于 Zag.js 状态机构建，所有交互组件都内置完整的 WAI-ARIA 支持，确保残障用户也能正常使用。

### 类型安全

使用 TypeScript 编写，提供完整的类型定义，在开发时获得准确的类型提示和错误检查。

### 样式灵活

使用 TailwindCSS v4 和 tailwind-variants 构建，支持主题定制和样式覆盖。

## 技术栈

| 技术 | 用途 |
|------|------|
| SolidJS | UI 框架 |
| Zag.js | 无头组件状态机 |
| TailwindCSS v4 | 样式系统 |
| tailwind-variants | 组件变体 |
| TypeScript | 类型系统 |

## 浏览器支持

支持所有现代浏览器：

- Chrome >= 90
- Firefox >= 90
- Safari >= 14
- Edge >= 90
