NavMenu 组件用于应用程序的导航，支持垂直（侧边栏）和水平（顶栏）两种布局。

## 基础用法

<NavMenuBasic client:load />

```tsx
import { NavMenu } from '@beeve/ui'
import { Home, LayoutDashboard, Settings } from 'lucide-solid'
import { createSignal } from 'solid-js'

const items = [
  { key: 'home', label: '首页', icon: <Home /> },
  { key: 'dashboard', label: '仪表盘', icon: <LayoutDashboard /> },
  { key: 'settings', label: '设置', icon: <Settings /> },
]

const [active, setActive] = createSignal('home')

<NavMenu
  items={items}
  value={active()}
  onValueChange={setActive}
/>
```

## 嵌套菜单

<NavMenuNested client:load />

```tsx
const items = [
  { key: 'home', label: '首页', icon: <Home /> },
  {
    key: 'settings',
    label: '设置',
    icon: <Settings />,
    children: [
      { key: 'profile', label: '个人资料' },
      { key: 'security', label: '安全设置' },
      {
        key: 'notifications',
        label: '通知设置',
        children: [
          { key: 'email', label: '邮件通知' },
          { key: 'push', label: '推送通知' },
        ],
      },
    ],
  },
]

<NavMenu
  items={items}
  value={active()}
  onValueChange={setActive}
  defaultExpandedKeys={['settings']}
/>
```

## 折叠模式

<NavMenuCollapsed client:load />

与 Sidebar 配合使用时，可以折叠为仅显示图标：

```tsx
const [collapsed, setCollapsed] = createSignal(false)

<NavMenu
  items={items}
  value={active()}
  onValueChange={setActive}
  collapsed={collapsed()}
/>
```

## 水平导航

<NavMenuHorizontal client:load />

```tsx
<NavMenu
  items={items}
  value={active()}
  onValueChange={setActive}
  direction="horizontal"
/>
```

## API

### NavMenuProps

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `items` | `NavMenuItemType[]` | **必填** | 菜单项数据 |
| `value` | `string` | - | 当前选中值（受控） |
| `defaultValue` | `string` | - | 默认选中值（非受控） |
| `onValueChange` | `(key: string) => void` | - | 选中值变化回调 |
| `expandedKeys` | `string[]` | - | 展开的菜单项 key（受控） |
| `defaultExpandedKeys` | `string[]` | - | 默认展开 key（非受控） |
| `onExpandedKeysChange` | `(keys: string[]) => void` | - | 展开变化回调 |
| `collapsed` | `boolean` | `false` | 是否折叠 |
| `direction` | `'vertical' \| 'horizontal'` | `'vertical'` | 方向 |

### NavMenuItemType

| 类型 | 属性 |
|------|------|
| 普通项 | `key`, `label`, `icon`, `badge`, `disabled`, `children`, `onClick` |
| 分隔线 | `{ type: 'divider' }` |
| 分组 | `{ type: 'group', key, label, children }` |
