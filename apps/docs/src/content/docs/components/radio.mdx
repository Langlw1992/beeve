Radio 组件用于在一组选项中进行单项选择。

## 基础用法

<RadioBasic client:load />

```tsx
import { Radio, Label } from '@beeve/ui'
import { createSignal } from 'solid-js'

const [value, setValue] = createSignal('option1')

<div class="flex flex-col gap-2">
  <div class="flex items-center gap-2">
    <Radio
      id="opt1"
      name="basic"
      value="option1"
      checked={value() === 'option1'}
      onChange={() => setValue('option1')}
    />
    <Label for="opt1">选项一</Label>
  </div>
  <div class="flex items-center gap-2">
    <Radio
      id="opt2"
      name="basic"
      value="option2"
      checked={value() === 'option2'}
      onChange={() => setValue('option2')}
    />
    <Label for="opt2">选项二</Label>
  </div>
</div>
```

## 禁用状态

<RadioDisabled client:load />

```tsx
<Radio disabled />                    {/* 禁用未选中 */}
<Radio disabled defaultChecked />     {/* 禁用已选中 */}
```

## API

### RadioProps

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `checked` | `boolean` | - | 是否选中（受控） |
| `defaultChecked` | `boolean` | `false` | 默认是否选中（非受控） |
| `onChange` | `(checked: boolean) => void` | - | 状态变化回调 |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | 单选框尺寸 |
| `disabled` | `boolean` | `false` | 禁用状态 |
| `id` | `string` | - | 元素 ID |
| `name` | `string` | - | 单选组名称 |
| `value` | `string` | - | 单选框值 |
