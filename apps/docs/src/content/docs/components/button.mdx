Button 组件用于触发一个操作，如提交表单、打开对话框等。

## 按钮变体

Button 支持多种视觉变体：

<ButtonVariants client:load />

```tsx
import { Button } from '@beeve/ui'

<Button variant="primary">主要按钮</Button>
<Button variant="secondary">次要按钮</Button>
<Button variant="outline">轮廓按钮</Button>
<Button variant="ghost">幽灵按钮</Button>
<Button variant="destructive">危险按钮</Button>
```

## 按钮尺寸

<ButtonSizes client:load />

```tsx
<Button size="sm">小按钮</Button>
<Button size="md">中按钮</Button>
<Button size="lg">大按钮</Button>
```

## 带图标按钮

<ButtonWithIcon client:load />

```tsx
import { Mail, ChevronRight } from 'lucide-solid'

<Button>
  <Mail class="mr-2 size-4" />
  发送邮件
</Button>

<Button variant="outline">
  下一步
  <ChevronRight class="ml-2 size-4" />
</Button>
```

## 加载状态

<ButtonLoading client:load />

```tsx
import { Loader2 } from 'lucide-solid'

<Button loading>
  <Loader2 class="mr-2 size-4 animate-spin" />
  加载中...
</Button>
```

## 禁用状态

<ButtonDisabled client:load />

```tsx
<Button disabled>禁用按钮</Button>
<Button variant="outline" disabled>禁用轮廓</Button>
```

## API

### ButtonProps

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `variant` | `'primary' \| 'secondary' \| 'outline' \| 'ghost' \| 'destructive' \| 'link'` | `'primary'` | 按钮变体 |
| `size` | `'sm' \| 'md' \| 'lg' \| 'icon'` | `'md'` | 按钮尺寸 |
| `loading` | `boolean` | `false` | 加载状态 |
| `disabled` | `boolean` | `false` | 禁用状态 |
| `type` | `'button' \| 'submit' \| 'reset'` | `'button'` | 按钮类型 |
| `class` | `string` | - | 自定义类名 |
| `onClick` | `(e: MouseEvent) => void` | - | 点击事件回调 |
