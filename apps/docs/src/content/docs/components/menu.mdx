Menu 组件用于显示下拉菜单和右键菜单，基于 Zag.js 实现。

## 下拉菜单

<MenuDropdown client:only="solid-js" />

```tsx
import { Dropdown } from '@beeve/ui'
import { SquarePen, Copy, Trash2 } from 'lucide-solid'

const items = [
  { key: 'edit', label: '编辑', icon: <SquarePen /> },
  { key: 'copy', label: '复制', icon: <Copy /> },
  { type: 'divider' },
  { key: 'delete', label: '删除', icon: <Trash2 />, danger: true },
]

<Dropdown items={items} onClick={(key) => console.log(key)}>
  <Button>操作</Button>
</Dropdown>
```

## 右键菜单

<MenuContextMenu client:only="solid-js" />

```tsx
import { ContextMenu } from '@beeve/ui'

<ContextMenu items={items} onClick={(key) => console.log(key)}>
  <div class="p-10 border rounded">
    右键点击此处
  </div>
</ContextMenu>
```

## API

### DropdownProps / ContextMenuProps

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `items` | `MenuItemType[]` | **必填** | 菜单项数据 |
| `children` | `JSX.Element` | **必填** | 触发元素 |
| `onClick` | `(key: string) => void` | - | 点击菜单项回调 |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | 尺寸 |
| `closeOnSelect` | `boolean` | `true` | 选择后是否关闭 |
| `positioning` | `PositioningOptions` | - | 定位选项 |
| `onOpenChange` | `(details: { open: boolean }) => void` | - | 打开状态变化回调 |

### MenuItemType

菜单项可以是以下类型：

| 类型 | 属性 |
|------|------|
| 普通项 | `key`, `label`, `icon`, `shortcut`, `disabled`, `danger`, `children`, `onClick` |
| 分隔线 | `{ type: 'divider' }` |
| 分组 | `{ type: 'group', label, children }` |
| 复选框 | `{ type: 'checkbox', key, label, checked, onChange }` |
| 单选组 | `{ type: 'radiogroup', name, value, children, onChange }` |
