Tooltip 组件用于在用户悬停时显示简短的提示信息，基于 Zag.js 实现。

## 基础用法

<TooltipBasic client:only="solid-js" />

```tsx
import { Tooltip, Button } from '@beeve/ui'

<Tooltip content="这是提示内容">
  <Button variant="outline">悬停查看提示</Button>
</Tooltip>
```

## 提示位置

<TooltipPositions client:only="solid-js" />

```tsx
<Tooltip content="顶部提示" positioning={{ placement: 'top' }}>
  <Button>上</Button>
</Tooltip>
<Tooltip content="右侧提示" positioning={{ placement: 'right' }}>
  <Button>右</Button>
</Tooltip>
<Tooltip content="底部提示" positioning={{ placement: 'bottom' }}>
  <Button>下</Button>
</Tooltip>
<Tooltip content="左侧提示" positioning={{ placement: 'left' }}>
  <Button>左</Button>
</Tooltip>
```

## 延迟显示

<TooltipWithDelay client:only="solid-js" />

```tsx
<Tooltip content="立即显示" openDelay={0}>
  <Button>无延迟</Button>
</Tooltip>
<Tooltip content="延迟显示" openDelay={500}>
  <Button>500ms 延迟</Button>
</Tooltip>
```

## 禁用提示

<TooltipDisabled client:only="solid-js" />

```tsx
<Tooltip content="不会显示" disabled>
  <Button>禁用提示</Button>
</Tooltip>
```

## 富文本提示

<TooltipRichContent client:only="solid-js" />

```tsx
<Tooltip
  content={
    <div class="space-y-1">
      <p class="font-medium">提示标题</p>
      <p class="text-xs text-muted-foreground">
        详细说明文本
      </p>
    </div>
  }
>
  <Button>富文本提示</Button>
</Tooltip>
```

## 带箭头提示

<TooltipWithArrow client:only="solid-js" />

通过 `arrow` 属性可以显示指向触发元素的箭头：

```tsx
<Tooltip content="带箭头的提示" arrow>
  <Button>默认箭头</Button>
</Tooltip>

<Tooltip content="顶部箭头" arrow positioning={{ placement: 'top' }}>
  <Button>上</Button>
</Tooltip>
```

## API

### TooltipProps

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `content` | `JSX.Element` | - | 提示内容 |
| `children` | `JSX.Element` | - | 触发元素 |
| `arrow` | `boolean` | `false` | 是否显示箭头 |
| `disabled` | `boolean` | `false` | 禁用提示 |
| `openDelay` | `number` | `300` | 显示延迟 (ms) |
| `closeDelay` | `number` | `0` | 隐藏延迟 (ms) |
| `positioning` | `PositioningOptions` | - | 定位选项 |
| `contentClass` | `string` | - | 自定义内容类名 |
| `onOpenChange` | `(details: { open: boolean }) => void` | - | 状态变化回调 |
