Switch 组件用于表示两种相互对立的状态间的切换。

## 基础用法

<SwitchBasic client:only="solid-js" />

```tsx
import { Switch, Label } from '@beeve/ui'
import { createSignal } from 'solid-js'

const [checked, setChecked] = createSignal(false)

<div class="flex items-center gap-3">
  <Switch
    id="basic"
    checked={checked()}
    onChange={setChecked}
  />
  <Label for="basic">开启通知</Label>
</div>
```

## 开关尺寸

<SwitchSizes client:only="solid-js" />

## 开关状态

<SwitchStates client:only="solid-js" />

```tsx
<Switch />                          {/* 关闭 */}
<Switch defaultChecked />           {/* 开启 */}
<Switch disabled />                 {/* 禁用关闭 */}
<Switch disabled defaultChecked />  {/* 禁用开启 */}
```

## API

### SwitchProps

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `checked` | `boolean` | - | 是否开启（受控） |
| `defaultChecked` | `boolean` | `false` | 默认是否开启（非受控） |
| `onChange` | `(checked: boolean) => void` | - | 开关状态变化回调 |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | 开关尺寸 |
| `disabled` | `boolean` | `false` | 禁用状态 |
| `id` | `string` | - | 元素 ID |
| `name` | `string` | - | 表单字段名 |
