Skeleton 组件用于在内容加载时展示占位效果，提供更好的加载体验。

## 基础用法

<SkeletonBasic client:only="solid-js" />

```tsx
import { Skeleton } from '@beeve/ui'

<Skeleton class="h-4 w-full" />
<Skeleton class="h-4 w-3/4" />
<Skeleton class="h-4 w-1/2" />
```

## 不同形状

<SkeletonShapes client:only="solid-js" />

```tsx
{/* 圆形 */}
<Skeleton class="size-12 rounded-full" />

{/* 方形 */}
<Skeleton class="size-12 rounded" />

{/* 矩形 */}
<Skeleton class="h-12 w-24 rounded" />
```

## 段落骨架屏

<SkeletonParagraph client:only="solid-js" />

```tsx
<Skeleton.Text rows={3} />
<Skeleton.Text rows={5} />
```

## 头像骨架屏

<SkeletonAvatar client:only="solid-js" />

```tsx
<Skeleton.Avatar size="sm" />
<Skeleton.Avatar size="md" />
<Skeleton.Avatar size="lg" />
<Skeleton.Avatar size={64} /> {/* 自定义尺寸 */}
```

## API

### SkeletonProps

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `width` | `string \| number` | - | 宽度 |
| `height` | `string \| number` | - | 高度 |
| `animated` | `boolean` | `true` | 是否显示动画 |
| `class` | `string` | - | 自定义类名 |

### Skeleton.Text Props

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `rows` | `number` | `3` | 行数 |
| `widths` | `(string \| number)[]` | - | 每行宽度 |

### Skeleton.Avatar Props

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `size` | `'sm' \| 'md' \| 'lg' \| number` | `'md'` | 尺寸 |
| `shape` | `'circle' \| 'square'` | `'circle'` | 形状 |
