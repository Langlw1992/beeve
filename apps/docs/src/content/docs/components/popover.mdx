Popover 组件用于显示浮动卡片，支持点击和悬停两种触发方式，可以包含任意内容，基于 Zag.js 实现。

## 基础用法

<PopoverBasic client:load />

```tsx
import { Popover, PopoverTitle, PopoverDescription, Button } from '@beeve/ui'

<Popover
  content={
    <div class="space-y-2">
      <PopoverTitle>气泡卡片标题</PopoverTitle>
      <PopoverDescription>这是气泡卡片的描述内容。</PopoverDescription>
    </div>
  }
>
  <Button variant="outline">点击打开</Button>
</Popover>
```

## 表单内容

Popover 可以包含复杂的交互内容，如表单。

<PopoverWithForm client:load />

```tsx
<Popover
  content={
    <div class="space-y-4">
      <PopoverTitle>编辑信息</PopoverTitle>
      <div class="space-y-2">
        <Label for="name">名称</Label>
        <Input id="name" placeholder="请输入名称" />
      </div>
      <div class="space-y-2">
        <Label for="email">邮箱</Label>
        <Input id="email" type="email" placeholder="请输入邮箱" />
      </div>
      <div class="flex justify-end gap-2">
        <Button size="sm" variant="outline">取消</Button>
        <Button size="sm">保存</Button>
      </div>
    </div>
  }
>
  <Button variant="outline">编辑</Button>
</Popover>
```

## 悬停触发

使用 `trigger="hover"` 开启悬停触发模式，鼠标移入时显示，移出后自动隐藏。

<PopoverHoverTrigger client:load />

```tsx
<Popover
  trigger="hover"
  arrow
  content={
    <div class="space-y-2">
      <PopoverTitle>悬停触发</PopoverTitle>
      <PopoverDescription>鼠标悬停时自动显示。</PopoverDescription>
    </div>
  }
>
  <Button variant="outline">悬停显示</Button>
</Popover>

{/* 自定义延迟 */}
<Popover
  trigger="hover"
  openDelay={0}
  closeDelay={0}
  content={<PopoverDescription>立即显示</PopoverDescription>}
>
  <Button variant="outline">无延迟</Button>
</Popover>
```

## API

### PopoverProps

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `content` | `JSX.Element` | - | Popover 内容 |
| `children` | `JSX.Element` | - | 触发元素 |
| `trigger` | `'click' \| 'hover'` | `'click'` | 触发方式 |
| `arrow` | `boolean` | `false` | 是否显示箭头 |
| `open` | `boolean` | - | 是否打开（受控） |
| `defaultOpen` | `boolean` | `false` | 默认是否打开 |
| `modal` | `boolean` | `false` | 是否为模态模式（仅 click） |
| `closeOnInteractOutside` | `boolean` | `true` | 点击外部是否关闭 |
| `closeOnEscape` | `boolean` | `true` | 按 ESC 是否关闭 |
| `openDelay` | `number` | `700` | 打开延迟 ms（仅 hover） |
| `closeDelay` | `number` | `300` | 关闭延迟 ms（仅 hover） |
| `positioning` | `PositioningOptions` | - | 定位选项 |
| `onOpenChange` | `(details: { open: boolean }) => void` | - | 状态变化回调 |
| `contentClass` | `string` | - | 自定义内容样式类名 |

### PopoverTitle

用于显示 Popover 标题的组件。

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `children` | `JSX.Element` | - | 标题内容 |
| `class` | `string` | - | 自定义类名 |

### PopoverDescription

用于显示 Popover 描述的组件。

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `children` | `JSX.Element` | - | 描述内容 |
| `class` | `string` | - | 自定义类名 |
