Input 组件用于接收用户输入的文本内容。

## 基础用法

<InputBasic client:load />

```tsx
import { Input } from '@beeve/ui'

<Input placeholder="请输入内容" />
```

## 输入框尺寸

<InputSizes client:load />

```tsx
<Input size="sm" placeholder="小尺寸" />
<Input size="md" placeholder="中尺寸" />
<Input size="lg" placeholder="大尺寸" />
```

## 密码输入框

<InputPassword client:load />

```tsx
import { Eye, EyeOff } from 'lucide-solid'
import { createSignal } from 'solid-js'

const [show, setShow] = createSignal(false)

<div class="relative">
  <Input
    type={show() ? 'text' : 'password'}
    placeholder="请输入密码"
  />
  <button onClick={() => setShow(!show())}>
    {show() ? <EyeOff /> : <Eye />}
  </button>
</div>
```

## API

### InputProps

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | 输入框尺寸 |
| `type` | `string` | `'text'` | 输入类型 |
| `placeholder` | `string` | - | 占位文本 |
| `value` | `string` | - | 输入值 |
| `disabled` | `boolean` | `false` | 禁用状态 |
| `readOnly` | `boolean` | `false` | 只读状态 |
| `class` | `string` | - | 自定义类名 |
| `onInput` | `(e: InputEvent) => void` | - | 输入事件回调 |
| `onChange` | `(e: Event) => void` | - | 变化事件回调 |
