Slider 组件用于在给定范围内选择一个值或范围，基于 Zag.js 实现。

## 基础用法

<SliderBasic client:load />

```tsx
import { Slider } from '@beeve/ui'
import { createSignal } from 'solid-js'

const [value, setValue] = createSignal([50])

<Slider
  value={value()}
  onValueChange={setValue}
  min={0}
  max={100}
/>
```

## 范围滑块

<SliderRange client:load />

```tsx
const [value, setValue] = createSignal([20, 80])
```

## 带标签滑块

<SliderWithLabel client:load />

```tsx
import { Label, Slider } from '@beeve/ui'

const [volume, setVolume] = createSignal([75])

<div class="space-y-2">
  <div class="flex items-center justify-between">
    <Label>音量</Label>
    <span class="text-sm text-muted-foreground">{volume()[0]}%</span>
  </div>
  <Slider value={volume()} onValueChange={setVolume} />
</div>
```

## API

### SliderProps

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `value` | `number[]` | - | 当前值（受控） |
| `defaultValue` | `number[]` | `[0]` | 默认值（非受控） |
| `onValueChange` | `(value: number[]) => void` | - | 值变化回调 |
| `min` | `number` | `0` | 最小值 |
| `max` | `number` | `100` | 最大值 |
| `step` | `number` | `1` | 步进值 |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | 滑块尺寸 |
| `disabled` | `boolean` | `false` | 禁用状态 |
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` | 方向 |
