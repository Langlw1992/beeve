import { BadgeBasic, BadgeDot, BadgeColors, BadgeStatus } from '../../../components/demos/BadgeDemo'

Badge 组件用于显示消息数量、状态指示等信息。

## 基础徽标

<BadgeBasic client:load />

```tsx
import { Badge } from '@beeve/ui'

<Badge count={5}>
  <Avatar />
</Badge>

<Badge count={99}>
  <Avatar />
</Badge>

<Badge count={100}>
  <Avatar />
</Badge>
```

## 小红点

<BadgeDot client:load />

```tsx
<Badge dot>
  <Bell />
</Badge>

<Badge dot color="blue">
  <Mail />
</Badge>
```

## 徽标颜色

<BadgeColors client:load />


## 状态徽标

<BadgeStatus client:load />

```tsx
<Badge status="default" text="默认" />
<Badge status="success" text="成功" />
<Badge status="processing" text="处理中" />
<Badge status="warning" text="警告" />
<Badge status="error" text="错误" />
```

## API

### BadgeProps

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `count` | `number \| JSX.Element` | - | 显示的数字或自定义内容 |
| `overflowCount` | `number` | `99` | 数字溢出时的最大值 |
| `dot` | `boolean` | `false` | 显示小红点 |
| `showZero` | `boolean` | `false` | 是否显示 0 |
| `color` | `'default' \| 'blue' \| 'green' \| 'orange' \| 'red'` | `'default'` | 徽标颜色 |
| `status` | `'default' \| 'success' \| 'processing' \| 'warning' \| 'error'` | - | 状态（用于独立状态点） |
| `text` | `string` | - | 状态文本（与 status 配合使用） |
| `size` | `'sm' \| 'md'` | `'md'` | 徽标尺寸 |
| `placement` | `'top-right' \| 'top-left' \| 'bottom-right' \| 'bottom-left'` | `'top-right'` | 徽标位置 |
| `offset` | `[number, number]` | - | 位置偏移 [x, y] |
| `children` | `JSX.Element` | - | 子元素 |
