Checkbox 组件用于在一组选项中进行多项选择。

## 基础用法

<CheckboxBasic client:load />

```tsx
import { Checkbox, Label } from '@beeve/ui'
import { createSignal } from 'solid-js'

const [checked, setChecked] = createSignal(false)

<div class="flex items-center gap-2">
  <Checkbox
    id="agree"
    checked={checked()}
    onChange={setChecked}
  />
  <Label for="agree">同意服务条款</Label>
</div>
```

## 复选框组

<CheckboxGroup client:load />

```tsx
const [selected, setSelected] = createSignal(['apple'])

const items = [
  { value: 'apple', label: '苹果' },
  { value: 'banana', label: '香蕉' },
  { value: 'orange', label: '橙子' },
]

const toggle = (value) => {
  setSelected((prev) =>
    prev.includes(value)
      ? prev.filter((v) => v !== value)
      : [...prev, value]
  )
}

{items.map((item) => (
  <div class="flex items-center gap-2">
    <Checkbox
      id={item.value}
      checked={selected().includes(item.value)}
      onChange={() => toggle(item.value)}
    />
    <Label for={item.value}>{item.label}</Label>
  </div>
))}
```

## API

### CheckboxProps

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `checked` | `boolean` | - | 是否选中（受控） |
| `defaultChecked` | `boolean` | `false` | 默认是否选中（非受控） |
| `onChange` | `(checked: boolean) => void` | - | 选中状态变化回调 |
| `indeterminate` | `boolean` | `false` | 不确定状态 |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | 复选框尺寸 |
| `disabled` | `boolean` | `false` | 禁用状态 |
| `id` | `string` | - | 元素 ID |
| `name` | `string` | - | 表单字段名 |
| `value` | `string` | - | 表单值 |
