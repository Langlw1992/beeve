Dialog 组件用于显示模态对话框，基于 Zag.js 实现。

## 基础用法

<DialogBasic client:load />

```tsx
import { Dialog, Button } from '@beeve/ui'
import { createSignal } from 'solid-js'

const [open, setOpen] = createSignal(false)

<Button onClick={() => setOpen(true)}>打开对话框</Button>
<Dialog
  open={open()}
  onOpenChange={setOpen}
  title="对话框标题"
  onOk={() => setOpen(false)}
  onCancel={() => setOpen(false)}
>
  <p>对话框内容</p>
</Dialog>
```

## 异步操作

<DialogAsync client:load />

当 `onOk` 返回 Promise 时，会自动显示加载状态：

```tsx
<Dialog
  open={open()}
  onOpenChange={setOpen}
  title="提交确认"
  onOk={async () => {
    await submitData()
    setOpen(false)
  }}
>
  <p>点击确定将模拟异步提交。</p>
</Dialog>
```


## API

### DialogProps

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `open` | `boolean` | - | 是否打开对话框 |
| `onOpenChange` | `(open: boolean) => void` | - | 打开状态变化回调 |
| `title` | `JSX.Element` | - | 对话框标题 |
| `description` | `JSX.Element` | - | 对话框描述 |
| `children` | `JSX.Element` | - | 对话框内容 |
| `footer` | `JSX.Element \| null` | - | 自定义底部，null 则不显示 |
| `okText` | `string` | `'确定'` | 确定按钮文字 |
| `cancelText` | `string` | `'取消'` | 取消按钮文字 |
| `okType` | `ButtonVariant` | `'primary'` | 确定按钮类型 |
| `onOk` | `() => void \| Promise<void>` | - | 点击确定回调 |
| `onCancel` | `() => void` | - | 点击取消回调 |
| `width` | `'sm' \| 'md' \| 'lg' \| 'xl' \| '2xl'` | `'lg'` | 对话框宽度 |
| `closable` | `boolean` | `true` | 是否显示关闭按钮 |
| `maskClosable` | `boolean` | `true` | 点击遮罩是否关闭 |
| `keyboard` | `boolean` | `true` | 按 ESC 是否关闭 |
