Label 组件用于为表单控件提供标签文本。

## 基础用法

<LabelBasic client:only="solid-js" />

```tsx
import { Label, Input } from '@beeve/ui'

<div class="space-y-2">
  <Label for="email">邮箱地址</Label>
  <Input id="email" type="email" placeholder="请输入邮箱" />
</div>
```

## 必填标记

<LabelRequired client:only="solid-js" />

```tsx
<Label for="username" required>用户名</Label>
<Input id="username" placeholder="请输入用户名" />
```

## 禁用状态

<LabelDisabled client:only="solid-js" />

```tsx
<Label for="disabled" disabled>禁用输入框</Label>
<Input id="disabled" disabled />
```

## 与表单控件配合

<LabelWithControls client:only="solid-js" />

```tsx
import { Label, Input, Checkbox, Switch } from '@beeve/ui'

{/* 带 Input */}
<div class="space-y-2">
  <Label for="name" required>姓名</Label>
  <Input id="name" placeholder="请输入姓名" />
</div>

{/* 带 Checkbox */}
<div class="flex items-center gap-2">
  <Checkbox id="agree" />
  <Label for="agree">我已阅读并同意服务条款</Label>
</div>

{/* 带 Switch */}
<div class="flex items-center justify-between">
  <Label for="notifications">接收通知</Label>
  <Switch id="notifications" />
</div>
```

## API

### LabelProps

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `for` | `string` | - | 关联的表单元素 ID |
| `required` | `boolean` | `false` | 是否必填（显示红色星号） |
| `disabled` | `boolean` | `false` | 是否禁用 |
| `class` | `string` | - | 自定义类名 |
| `children` | `JSX.Element` | - | 标签文本 |
